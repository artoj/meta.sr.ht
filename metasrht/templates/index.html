git.sr.ht

git.sr.ht is our git repository hosting service.

  • Public, private, and "unlisted" repositories
  • Fine grained access control, including access for users without accounts
  • Software release management

builds.sr.ht

builds.sr.ht is our continuous integration service.

  • Runs fully virtualized builds on various Linux distros and BSDs
  • Run anonymous jobs that aren't tied to a repository
  • Post-build triggers for email, webhooks, etc
  • Powerful API for complex build scenarios

lists.sr.ht

lists.sr.ht is our powerful mailing list service.

  • Mail submission and patch review via the web
  • Threaded, searchable mail archives
  • Tools for working with third party mailing lists

todo.sr.ht

todo.sr.ht is our ticket tracking service.

  • Customizable ticket flows to suit your project
  • Private bug reports and bug trackers for security issues
  • Fine-grained access controls
  • Separate personal and public custom ticket filters

dispatch.sr.ht

dispatch.sr.ht is our task dispatcher and service integration tool.

  • Automate tasks between various sr.ht services
  • Integrate sr.ht with third-party services like GitHub
  • Reduces the workload for maintainers and contributors

man.sr.ht

man.sr.ht is our markdown- and git-driven wiki service.

  • Use git to version control and manage your wiki
  • Use any organizational hierarchy you like, a flat wiki is not imposed
  • Hosts the detailed sr.ht manual

meta.sr.ht

meta.sr.ht is our account management service.

  • PGP encrypted and signed emails from sr.ht services
  • Two-factor authentication with TOTP
  • Detailed audit logs of account activity
  • Fine-grained third-party OAuth access controls
